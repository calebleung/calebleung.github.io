Dive Master

The wiki for Guild Wars 2 is incredibly extensive. Easily accessed from in-game, it is an excellent resource for its players. However, as it is a wiki for all players, none of this information is personalized. This can be an issue for multi-step achievements such as Dive Master. Further, the in-game description of each piece of the Dive Master achievement simply tells the player which diving spot they're missing, #1, #3, #n.

I created the Dive Master browser extension to alleviate this problem. Using the game's API, I added a personalized column into the Dive Master wiki page which indicates that player's progress with the achievement. Combining the API's knowledge of the player's completion of each diving spot and pairing it with the wiki's description of each spot, a player has a more complete picture of where to go to finish the Dive Master achievement.

Link Shortner

Have you ever had an idea stuck in your head that wouldn't go away until you executed on it? The Link Shortner project was just that. As the name suggests, it is a link shortner proof-of-concept. Shortlinks for http(s) are created via GETs and POSTs and uses Google Safe Browsing to safeguard against being used to send users to malicious content.

Takeout Text Reader

As a companion project to Takeout to SMS Backup & Restore, Takeout Text Reader takes the same data and displays texts in a chat-esque format. The two projects use data from Google Takeout to preserve texts from the Google Voice service. This is especially important as participating in Google's Project Fi prevents users from accessing Google Voice at all. As such, being able to export out texts is highly desirable.
